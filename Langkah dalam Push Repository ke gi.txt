Langkah dalam Push Repository ke gitHub dengan SSH :

1. dalam git bash ketikan perintah < ssh-keygen > yang berfungsi untuk membuat key, kemudian enter saja semua prosesnya. Setelah itu gunakan perintah < cat > diikuti dengan direktory tempat file .pub nya kemudian enter dan key akan secara otomatis di generate.
2. copy key tersebut kemudian buka settingan ssh di github dan klik new ssh dan masukkan key yang sudah di copy tersebut, kemudian save.
3. Langkah berikutnya yaitu ketikan perintah :
	a. git init
	b. git add .
	c. git branch -M main
	d. git commit -m " Komit pertama! "
	e. git remote add origin < kode ssh dari github >
	f. git push -u origin main
